# Jogo-da-forca
Voltado ao estudo e diverção. 
Tentado sair um pouco do comum que é fazxer algo voltado a empresas, decidi usar o que sei para fazer um jogo.
Simples e facil de entender, nasceu a ideia do jogo da forca interativo.

Usei módulos, Padrão ASIC, Linguagem C.
Strings, biblioteca randomizadora.

Foi usado uma listagem de temas e palavras pré estabelecidas.
